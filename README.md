# OLSheets
**OLSheets** is a web-based marketplace that enables musicians and other members of the general public to **rent instruments and music sheets** from one another, with built-in search, booking and secure payment. Owners are able to list and manage items for rent, while renters can search, filter, book and perform payments through the platform.
